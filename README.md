PG Spotter is a web-based application developed to help users easily search and find paying guest (PG) accommodations based on their location and property preferences. The system allows users to browse different PG listings and view detailed information about available accommodations, including amenities, pricing, and location details.

The main objective of this project is to simplify the process of finding suitable PG housing for students or individuals who are moving to a new city. Users can search for PG properties by selecting a state or entering the name of the PG. The system then retrieves relevant information from the database and displays a list of available properties. Each listing includes important details such as the PG title, facilities like air conditioning, balcony availability, parking, furnished rooms, and monthly rent. This helps users compare options and make better decisions when choosing a place to stay.

The application uses dynamic web pages to fetch and display data stored in a database. Property information is stored in tables and retrieved using server-side scripts whenever users perform a search. The system also includes structured pages such as a homepage, property listing section, property details page, and common components like header and footer sections to maintain a consistent layout throughout the website.

This project demonstrates practical implementation of full-stack web development concepts, including database connectivity, server-side processing, and responsive user interface design. The front-end design is built to be simple and user-friendly so that users can easily navigate through the website and access property information quickly.

PG Spotter was developed as part of an academic project for a Bachelor of Computer Applications program and showcases the use of modern web development technologies for building database-driven applications.
