PG Spotter – PG Accommodation Finder
Overview

PG Spotter is a web-based application developed to help students and working professionals easily find Paying Guest (PG) accommodations. The platform allows users to browse available PG listings, view detailed information about accommodations, and contact property owners. An admin panel is also provided to manage listings and maintain the system efficiently.

This project was developed as part of my Bachelor of Computer Applications (BCA) coursework to demonstrate practical skills in web development using PHP and MySQL.

Features:
User registration and login system
Browse available PG accommodations
View detailed PG information
Contact PG owners
Admin dashboard for managing listings
Add, update, and delete PG records
Responsive web interface
